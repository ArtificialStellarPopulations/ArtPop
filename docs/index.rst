ArtPop
######

.. raw:: html

   <a href="https://www.spacetelescope.org/news/heic1617/">
       <img src="_static/Terzan-5.jpeg" width="100%" alt="Terzan 5 Cluster"/>
   </a>

**Art**\ ificial Stellar **Pop**\ ulations --- `ArtPop` --- is a 
`Python <https://www.python.org>`_ software package for synthesizing stellar 
populations and simulating realistic images of stellar systems. The code is designed to be 
intuitive and as modular as possible, making it possible to use each of 
its functionalities independently or together.
